## Installation

### Verify `node.js` and `npm` installation
```js
node -v
npm -v
```

### dotenv
```js
npm install dotenv
```

### Initialize Node & Install Dependencies 
```sh
npm init -y                 
npm i express dotenv
```
> 1. `npm init -y`: Initialize project
> 2. `npm i express dotenv`: Initialize the `dotenv` package
